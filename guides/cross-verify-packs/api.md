# API Domain Questions

Additional cross-verify questions for API development (REST, GraphQL, MCP).

| #   | Dimension | Question                                                               |
| --- | --------- | ---------------------------------------------------------------------- |
| A1  | Body      | Are all endpoints input-validated (type, length, format)?              |
| A2  | Body      | Are breaking changes versioned or behind feature flags?                |
| A3  | Mind      | Is the API contract documented (OpenAPI, GraphQL schema)?              |
| A4  | Heart     | Do error responses include actionable guidance, not just status codes? |
| A5  | Spirit    | Is rate limiting configured on public endpoints?                       |
