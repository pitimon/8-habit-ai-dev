# Mobile Domain Questions

Additional cross-verify questions for mobile app development (iOS, Android, React Native, Flutter).

Sources: Apple HIG, Material Design, WCAG mobile accessibility guidelines.

| #   | Dimension | Question                                                                   |
| --- | --------- | -------------------------------------------------------------------------- |
| B1  | Body      | Does the app handle offline and poor connectivity gracefully?              |
| B2  | Body      | Is battery and memory impact acceptable (no unnecessary background drain)? |
| B3  | Heart     | Are touch targets at least 44x44pt (accessibility, Apple HIG)?             |
| B4  | Mind      | Is responsive layout tested across screen sizes and orientations?          |
| B5  | Spirit    | Does the app comply with App Store and Play Store review guidelines?       |
