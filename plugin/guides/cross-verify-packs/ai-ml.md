# AI/ML Domain Questions

Additional cross-verify questions for AI/ML development (models, pipelines, inference).

Sources: Google PAIR Guidebook, Microsoft Responsible AI, OECD AI Principles.

| #   | Dimension | Question                                                                  |
| --- | --------- | ------------------------------------------------------------------------- |
| M1  | Body      | Is model versioning in place (reproducible training and inference)?       |
| M2  | Mind      | Is the training data pipeline validated (no data leakage, no label bias)? |
| M3  | Heart     | Are model outputs explainable to end users (not just developers)?         |
| M4  | Spirit    | Has bias been assessed across demographic groups (fairness metrics)?      |
| M5  | Spirit    | Is prompt injection prevention in place for LLM-based features?           |
